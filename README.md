# Gex, Enter the Ginkgo

Makes the Ginkgo interface more useable by increasing the text area and adding keyboard navigation.

Keyboard controls:
* **Play/Pause** 
  * K
  * \<space\>
* **Next Slide**
  * N
  * \<right arrow\>
  * \<period\>
  * \<enter\>
* **Previous Slide**
  * P
  * \<left arrow\>
* **Rewind**
  * J
* **Fast Forward**
  * L

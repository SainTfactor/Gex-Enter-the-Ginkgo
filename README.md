# Gex, Enter the Ginkgo

Makes the Ginkgo interface more useable by increasing the size of the gray text area at the bottom and adding keyboard navigation.

Keyboard controls:
* **Play/Pause** 
  * K
  * \<space\>
* **Next Slide**
  * N
  * \<right arrow\>
  * \<period\>
  * \<enter\>
* **Previous Slide**
  * P
  * \<left arrow\>
* **Rewind**
  * J
* **Fast Forward**
  * L
* **Font Size Up**
  * W
  * \<plus\>
* **Font Size Down**
  * Q
  * \<minus\>
